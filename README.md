# Jose Antonio Cuenca - Distribuidor de Sistemas de Descanso

Web moderna y elegante para distribuidor de sistemas de descanso premium.

## Características

- Diseño moderno y elegante con colores negros, grises y marrones
- Totalmente responsive (móvil, tablet, desktop)
- Animaciones suaves y transiciones profesionales
- Sección de marcas con enlaces a proveedores oficiales:
  - TORRESOL
  - KOALA
  - KARIBIAN
  - IVORIMATEX
  - B-SENSIBLE
- Galería de productos interactiva
- Formulario de contacto
- SEO optimizado

## Tecnologías Utilizadas

- **React 18** - Framework de UI
- **TypeScript** - Tipado estático
- **Vite** - Build tool ultrarrápido
- **Tailwind CSS** - Framework CSS utility-first
- **Lucide React** - Iconos modernos
- **Framer Motion** - Animaciones (preparado para usar)

## Instalación

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Vista previa de la build de producción
npm run preview
```

## Desarrollo

El servidor de desarrollo se ejecuta en http://localhost:5173
